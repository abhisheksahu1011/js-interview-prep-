            What are the differences between var, let and const?
            => var is a Global or Functional Scoped.it can be reassigned or redeclared and it can be modified later.
            =>let is a Block Scoped.it cannot be reassigned or redeclare. let can be modified later.
            =>const is a Block Scoped.it cannot be reassigned or redeclare.const cannot be modified later.
            
        
            What is execution context
            => A special environment to handle the transformation and execution of this JavaScript code. This environment is known as the Execution Context.

            During the Execution Context run-time, the specific code gets parsed by a parser, the variables and functions are stored in memory, executable byte-code gets generated, and the code gets executed.
                There are two kinds of Execution Context in JavaScript:
                ●	Global Execution Context (GEC)
                ●	Function Execution Context (FEC)
